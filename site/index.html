RL ProtoKit: Punk Your RL Workflow!

RL ProtoKit is a lightweight CLI tool for reinforcement learning developers. It helps with quick prototyping by generating custom env wrappers, tuning hyperparameters, debugging policies, and running full pipelines. Built to stand out in job apps, it's easy to install and use. Run pip install -e . in the root, then try commands in the terminal on the right.

Generate Command

Creates custom Gym wrappers. Components: Parse mods (splits input), Build template (constructs class), Output file (writes code). Final output: .py file with Wrapper class. Use for env mods like reward scaling in CartPole to stabilize training.

Tune Command

Runs grid search. Components: Create grid (generates combos), Run trials (parallel training), Rank & save (selects best). Final output: tuned_model.pth. Great for optimizing learning rates in DQN models.

Debug Command

Steps through policies. Components: Load model (torch.load), Simulate steps (env.step), Display values (Rich table). Final output: Decision table. Helps spot issues in agent behavior.

Full Pipeline

Chains all commands. Components: Generate, Tune, Debug. Final output: Complete results. Ideal for end-to-end testing.
