RL ProtoKit: Punk Your RL Workflow!

RL ProtoKit is a CLI toolkit for RL developers. Instantly generate custom wrappers, tune hyperparameters, debug policies, and run full pipelines‚Äîall with a punk attitude.
Install with pip install -e . and try the commands in the CRT terminal on the right.

Generate Command

  • Purpose: Quickly create custom Gym wrappers.
  • How: Parses mods, builds a Python class, outputs a ready-to-import file.
  • Example: 
    class CustomWrapper(gym.Wrapper):
    def step(self, action):
        obs, reward, terminated, truncated, info = self.env.step(action)
        reward *= 0.5
        return obs, reward, terminated, truncated, info

Tune Command

  • Purpose: Grid search for RL hyperparameters.
  • How: Creates grid, runs parallel trials, ranks and saves best model.
  • Output: tuned_model.pth

Debug Command

  • Purpose: Step through and visualize a trained RL policy.
  • How: Loads model, simulates steps, displays Q-values and confidence.
  • Output: Table of state, action, Q-values.

Full Pipeline

  • Purpose: Run the entire workflow: generate ‚Üí tune ‚Üí debug.
  • How: Chains all commands, outputs results.
